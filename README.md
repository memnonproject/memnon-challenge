# memnon-challenge
Memnon Challenge

The Memnon Challenge is an open design challenge exploring systems that use solar thermal energy to generate musical sculpture and performance.

The project draws inspiration from the Vocal Memnon and other historical precedents, thermoacoustic engineering, moving shadows and the sun-earth relationship, landscape art and kinetic art.
