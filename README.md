# memnon-competition
Memnon Competition

The Memnon Competition is a proposed open design challenge exploring systems that use solar energy both to generate sound and to control musical performance.

The project draws inspiration from the historical Vocal Memnon, solar thermal music research, thermoacoustics, kinetic sculpture, and experimental musical instrument design.
